Building Timings
================


This utility prints out timings of the first important buildings across
many games.


Important buildings:

* nexus
* hatch
* cc
* spawning pool
* spire
* gateway
* baracks


requirements
------------

Uses the ggtracker fork of sc2reader
Assumes a ``replays`` directory exists with many replays in it (searches recursively)


use
---

python timings.py
